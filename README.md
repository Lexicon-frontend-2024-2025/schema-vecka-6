## Inför presentation/redovisning
Förbered en 15 min presentation av ert projekt där ni demar den centrala funktionalitet ni har gjort och berättar kortfattat om koden bakom denna. Använd er av länkarna nedan och se till att kunna dela koden via Live Share i VS Code så att andra kan följa med ordentligt.
